### git commit message 규칙 (나만의 규칙)
- 파일 생성 `✨new : ~~~ 생성에 대한 커밋`
- 파일 업데이트 `🚀update : ~~~ 추가에 대한 커밋`
- 파일 수정 `✂fix : ~~~ 수정에 대한 커밋`
- 파일 삭제 `🗑delete : ~~~ 삭제에 대한 커밋`
