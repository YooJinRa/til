# MIME Type
- 클라이언트에게 전송된 문서의 다양성을 알려주기 위한 메커니즘
- 웹에서 파일의 확장자는 별 의미가 없음 => 그 문서와 함께 올바른 MIME 타입을 전송하도록, 서버가 정확히 설정하는 것이 중요
- 브라우저들은 리소스를 내려받았을 때 해야 할 기본 동작이 무엇인지를 결정하기 위해 대게 MIME 타입을 사용
